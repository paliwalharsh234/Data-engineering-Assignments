# Superstore Sales Analysis using SQL

## Project Objective
Analyze Superstore sales data using SQL to extract business insights through filtering, aggregation, trend analysis, and data quality validation.

## Tools Used
- MySQL Workbench
- SQL
- Superstore Sales Dataset

## Key Analysis Performed
- Data exploration
- Filtering using WHERE clause
- Aggregations using GROUP BY
- Top products and customers analysis
- Monthly sales trend analysis
- Duplicate record detection
- Data quality checks

## Key Insights
- Identified top revenue-generating customers.
- Found best-selling products and categories.
- Analyzed monthly sales trends.
- Validated dataset quality by checking duplicates and null values.
